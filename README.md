# 📓 MyJournal

![Logo]

**MyJournal** is a simple and clean journaling application that allows users to create, manage, and save their daily entries easily. It helps users organize thoughts, track memories, and maintain a personal digital diary.

---

## 🚀 Features

✨ Easy-to-use interface
📝 Add journal entries quickly
💾 Save entries securely
📂 Organize daily thoughts efficiently
📱 Clean and minimal home screen
